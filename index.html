Ellie Beauty House

//2018

view project source code
  • HTML5
  • CSS3
  • JavaScript
  • jQuery
  • Responsive
  • Bootstrap

Ellie Beauty House is a beauty shop that specialize in Microblading, Permanent Makeup and Eyelash Extensions. The client required functionality such as being fully responsive, easy to navigate, a beautiful UI, and a gallery to showcase thier work.

The client requested that the website consisted of 4 web pages that would look beautiful on any device. I focused on creating a simple and easy to read pricing table for customers to get infomation. My goal was to build something that thier customers could easily find and understand the services they provide in order to book an appointment.

From time to time the client would return to me with additional updates or new problems that needed solving.
